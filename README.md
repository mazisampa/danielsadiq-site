# danielsadiq.com

A fast, editorial personal website for Daniel Sadiq.

## What is included
- Home page
- Writing index
- Four launch essays
- About page
- Work With Me page
- Responsive layout
- SEO titles and descriptions
- Email-based inquiry and subscription flows

## Local preview
Run this command in the project folder:

```bash
python3 -m http.server 8080
```

Open http://localhost:8080.

## Fastest deployment: Vercel
1. Import this repository into Vercel.
2. Deploy with no build command.
3. Add `danielsadiq.com` and `www.danielsadiq.com` under Project Settings > Domains.
4. Follow Vercel's DNS instructions at the registrar where the domain is registered.
